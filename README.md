# 🌟 별 수집 가챠 게임 (Star Gatcha)

실제 천문학 데이터(HYG Database)를 활용하여 별을 수집하는 게임 프로젝트입니다.

## 🌌 프로젝트 개요
이 프로젝트는 약 12만 개의 실제 별 데이터를 기반으로 플레이어가 랜덤 확률로 별을 수집하는 게임입니다.
수집 이후 어떤 컨텐츠를 추가할지는 고민 중...

## 💎 별 등급 분류 기준
본 게임에서는 별의 희귀도를 다음과 같은 기준으로 나눕니다 (등급 명칭은 추후 확정 예정):

| 분류 | 조건 | 데이터 특징 | 개수 |
| :--- | :--- | :--- | :--- |
| **최고 등급** | 고유 이름 존재 | `proper` 컬럼에 명칭 있음 (예: 시리우스, 베가) | 499개 |
| **중간 등급** | 천문 기호 존재 | `proper`는 없으나 `bf` 컬럼에 값이 있음 (예: 21Alp And) | ~2,600개 |
| **일반 등급** | 번호로 식별 | `proper`와 `bf` 모두 없음 (번호로만 불리는 별들) | ~116,500개 |

## 🛠️ 개발 환경 및 폴더 구조
- **Main Development**: Unity (C#)
- **Data Analysis**: Python (Pandas)

### 폴더 구조
- `Data/`: 원본 CSV 및 가공된 데이터 파일
- `Python/`: 데이터 전처리 및 분석 스크립트 (개발 예정)
- `Unity/`: 유니티 프로젝트 소스 (시작 예정)
- `Docs/`: 프로젝트 관련 문서 및 샘플 파일
